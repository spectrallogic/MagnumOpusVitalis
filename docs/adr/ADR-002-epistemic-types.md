# ADR-002 — Epistemic types

Status: accepted · Era 6 · applies to magnum_opus_v2 AND primordium

## Decision

Every piece of remembered or recalled content carries an **epistemic
type**. Types are not advisory: they determine what the content may
become. The core law: **imagination never masquerades as observation.**

## The tag set

| Tag | Meaning | Examples in this repo |
|---|---|---|
| `observed` | direct sensor/outcome evidence | Memory `_capture` traces, Chronicle episodes, Reach bank entries |
| `inferred` | derived by the runtime's own models from observed evidence | (future: belief-graph derivations) |
| `reported` | asserted by another agent | user text, Foster replies (stored in Wordstream transcript with source) |
| `recalled` | a retrieval/reconstruction of an earlier trace | Reach retrievals, `_latent_recall` matches |
| `imagined` | generated by the mind's own predictive machinery | mind's-eye stream (`minds_eye:{prediction,dream}`, conf 0.8), Wheel expectations, speculation rollouts |
| `confabulated` | deliberately synthesized false memory | FalseMemoryConfabulator traces (`epistemic="confabulated"`, confidence<1, `false=True`) |

## Eligibility: what each type may become

| Destination | observed | inferred | reported | recalled | imagined | confabulated |
|---|---|---|---|---|---|---|
| Canonical artifacts (abstraction ladder, bus attractors, semantic consolidation) | YES | future: with confidence | future: with trust | only if the source trace was observed | **NO** | **NO** |
| Self-model "what-just-was" leak | YES | no | no | no | **NO** | **NO** |
| Subconscious sea / intrusive thoughts | yes | yes | yes | yes | yes (tagged) | yes (down-weighted — deliberate design: the dreaming mind cannot tell) |
| Training targets (world model) | YES | no | via observed channels only | no | **NO** (dreams are never trained on — long-standing rule) | **NO** |
| UI display | yes, and the tag is shown where it matters (recall shows `false`) | — | — | — | — | — |

Chosen policy (Era 6): **guard canonical only.** Confabulation and
imagination keep their roles in the undertow — marked, down-weighted,
deliberately indistinguishable to the dreaming layers — but are blocked
from every path that produces a persistent belief-like artifact. The
contamination test asserts BOTH directions: the block holds, and the
undertow still dreams.

## Action/efference audit (blueprint P1)

Primordium's world model is action-conditioned at the input: PROPRIO = 26
dims = voice 12 (11 motor + self_heard) + keys 3 + paint 8 + gaze 3,
entering the token group with a ONE-TICK delay (efference of tick t is an
input at tick t+1's group; traces pair action t with reward at t+1 for
motor credit). The falsifiable proof that this conditioning is load-bearing
is the zeroed-efference canvas probe (Era 6 M5 test).
